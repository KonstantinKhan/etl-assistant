## Dependencies

## Testing

- Use Kotest, Should Spec test style

## Building & Running

To build or run the project, use one of the following tasks:
| Task | Description |
|-------------------|------------------|
| `./gradlew test`  | Run the tests |
| `./gradlew build` | Build everything |
| `./gradlew run`   | Run the server |
